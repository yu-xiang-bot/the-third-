# 双锋对决 项目文件功能说明

## 1. 项目概述

"双锋对决" 是一款使用HTML5+JavaScript编写的2D横版双人格斗类本地网页游戏。项目包含完整的游戏逻辑、界面设计、资源文件和文档说明。

## 2. 文件分类与功能说明

### 2.1 主要HTML文件

#### 2.1.1 index.html (11.56 KB)
- **功能**：游戏主界面文件
- **内容**：
  - 游戏主体结构（地图、角色、UI元素）
  - 音频资源引用（法师、机械师、背景音乐等）
  - 关卡选择菜单和帮助说明模态框
  - 游戏控制按钮（主菜单、音效开关、帮助等）
- **引用资源**：new.css、jquery.min.js、control.js

#### 2.1.2 get_image_size.html (700 B)
- **功能**：辅助工具页面，用于获取图片尺寸
- **内容**：
  - JavaScript代码，用于加载和检测图片尺寸
  - 主要用于开发阶段测试背景图片尺寸
- **目标图片**：img/steve.art/background.png

### 2.2 JavaScript文件

#### 2.2.1 control.js (64.55 KB)
- **功能**：游戏核心逻辑和控制
- **主要模块**：
  - 碰撞检测系统 (`collisionCheak` 函数)
  - 随机数生成 (`randomNumberAtoB` 函数)
  - 游戏主逻辑 (`game` 函数)
  - 角色控制类 (`Control1` 和 `Control2`)
  - 关卡系统管理
  - 技能系统实现
  - 能量和生命值管理
  - 音效系统控制
- **重要特性**：
  - 实现了完整的物理引擎（重力、速度、加速度）
  - 复杂的技能冷却机制
  - AI对战模式支持
  - 多关卡背景切换

#### 2.2.2 jquery.min.js (84.68 KB)
- **功能**：jQuery库文件
- **版本**：压缩版jQuery库
- **用途**：提供DOM操作、事件处理等基础功能

## 4. 运行流程

1. **启动阶段**：
   - 用户访问 `index.html`（主入口页）
   - 加载主界面样式和交互逻辑
   - 显示游戏开始按钮和操作说明
   - 初始化游戏逻辑（`control.js`）

3. **游戏执行**：
   - `control.js` 主循环运行
   - 处理用户输入、碰撞检测、技能释放等
   - 根据游戏状态播放相应音效和显示动画

4. **辅助工具**：
   - `get_image_size.html` 可用于开发过程中测试图片尺寸

## 5. 技术特点

1. **纯前端实现**：使用HTML5+CSS3+JavaScript，无需后端服务器
2. **模块化设计**：功能分离，代码结构清晰
3. **丰富资源**：包含完整的图像和音频资源
4. **响应式设计**：适应不同屏幕尺寸
5. **国际化支持**：中英文切换功能
6. **性能优化**：使用压缩版jQuery，优化资源加载

## 6. 总结

"双锋对决"项目是一个结构完整、功能丰富的双人格斗游戏，包含了从游戏逻辑到用户界面的全部实现。项目文件组织清晰，功能模块化设计良好，是HTML5游戏开发的优秀示例。通过详细的资源文件和完善的文档，开发者可以轻松理解项目结构和功能实现，也便于后续的维护和扩展。
